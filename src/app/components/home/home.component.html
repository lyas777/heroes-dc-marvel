Comic App

¡Bienvenido al Mundo del Cómic! Este proyecto es ejemplo de uso de Angular para realizar una página Web sobre los héroes del mundo de los cómics. En la sección de héroes, se muestra una lista de héroes del mundo Marvel y DC. Así mismo, en el navbar, se puede econtrar un buscador dónde se puede ingresar el nombre de algún héroe para encontrar su información.
